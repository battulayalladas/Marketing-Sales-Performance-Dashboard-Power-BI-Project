# 📊 Marketing & Sales Performance Dashboard (Power BI Project)

## 📌 Project Overview

This project focuses on analyzing **marketing and sales performance** using key business metrics such as campaign performance, billing trends, revenue generation, and learning program effectiveness.

The goal of this project is to provide **data-driven insights** to improve business decisions and optimize marketing strategies using an **interactive Power BI dashboard**.

---

## 🎯 Problem Statement

Organizations invest heavily in marketing campaigns and learning programs, but often lack clear visibility into:

* Which campaigns generate the most revenue
* How sales vary across locations and time
* Which learning programs perform best
* How billing trends impact overall earnings

This project aims to solve these challenges through data analysis and visualization.

---
